## 常用功能
###Components
* 所有配置文件 
  Components ->  all config files
  
* 所有系统参数 
  Components ->  当前系统环境参数 & 当前系统参数
  
* 应用基本信息(查看文件句柄数)
  Components -> Host info 
  
* JVM 内存实时监控 
Components -> JVM Memory Pool monitor 

###Analyzer
* jar依赖情况 
  Analyzer->jar dependencies
  
* java 内存对象
  Analyzer->JVM Sampler
  
* gc详细情况
  Analyzer->VisualGC  
  
###Thread dump
查看当前线程情况

###File Log
查看日志情况
